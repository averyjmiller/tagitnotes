# Tag-it Notes

Tag-it Notes is a full-stack note-taking application built with the **MERN stack** (MongoDB, Express, React, Node.js).  
It allows users to create, edit, and delete notes, organize them with **tags**, and filter notes for easy access.  

## Description

This project demonstrates:
- User authentication (signup, login, JWT)
- CRUD functionality for notes
- Tagging system with filter/search capabilities
- Responsive frontend with React
- Backend API with Express and MongoDB Atlas

Deployed frontend and backend (links to Vercel / Render once live)  

Tag-it Notes is perfect for anyone looking to organize their thoughts, tasks, or ideas efficiently.
